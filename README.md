# Bellabeat Case Study

## Project Overview

This project analyzes smart device usage data from Fitbit users to identify trends in physical activity and sleep behaviors. The goal is to provide business recommendations that can help Bellabeat improve its marketing strategy and user engagement.

---

## Business Task

Analyze smart device usage data to identify trends in user behavior and provide recommendations for Bellabeat's marketing strategy.

---

## Dataset

- Source: Fitbit Fitness Tracker Data (Kaggle)
- Users: 33
- Period: One month

Files used:

- dailyActivity_merged.csv
- sleepDay_merged.csv

## Key Findings

- Daily steps have a moderate positive relationship with calories burned.
- Sleep duration has a weak relationship with daily physical activity.
- Most users are classified as Low Active or Somewhat Active.
- No clear trend shows that more active users sleep longer.

---

## Recommendations

- Encourage inactive users through personalized step goals.
- Introduce activity challenges and motivational rewards.
- Continue promoting healthy sleep habits.
- Provide personalized health recommendations.
